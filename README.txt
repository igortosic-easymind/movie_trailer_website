Movie trailer website

1. Download the repository and start index.html with your favorite browser

or if you have installed Python:

2. Start your terminal and cd to your directory with movie trailer website and type command:

python -m SimpleHTTPServer 8080

- This command just created your local server with a port number of 8080. 
  This can be accessed via your browser at http://localhost:8080/

if you are familliar with npm - follow this:

3. Clone the repository and install with npm
   
    npm install -g http-server

    Then simply run http-server in any of your project directories:
    Eg. d:\my_project> http-server
